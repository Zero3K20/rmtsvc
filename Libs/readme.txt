The bin directory contains the lib library files required to compile rmtsvc:
jpeg-d.lib       - JPEG library compiled in debug multithreaded mode
jpeg-r.lib       - JPEG library compiled in release multithreaded mode
jpeg-r-dll.lib   - JPEG library compiled in release multithreaded DLL mode

net4cpp21.lib          - Network library compiled in release multithreaded DLL mode
net4cpp21_nossl.lib    - Network library compiled in release multithreaded DLL mode without SSL encryption support

vIDC.lib               - vidc library compiled in release multithreaded DLL mode
